# DOM-ejercicios
EJERCICIO 1
- Dentro de una función llamada contarEnlaces()
- Cree una variable arregloEnlaces que obtiene todos los Tags de tipo <a> que encuentre en el documento y los guardará como array
- Después cree una variable con nombre largo que guarda el total de elementos mediante .length
- La variable númeroDeEnlaces obtendrá la sección de html donde se escribirá el resultado
- Finalmente, mediante innerHTML en la sección obtenida del html se escribe el total del número de enlaces.

EJERCICIO2
- Dentro de una función llamada poquitosOmuchos()
- Cree la variable elememtosMensaje que guarda en un array todos los elementos de tipo clase con el nombre "mensaje"
- Después cree una variable con nombre noElementosMensaje que guarda el total de elementos mediante .length
- Finalmente para valorar el número de elementos utilicé el condicional if, en caso de encontrar un sólo elemento mostrará un alert con el mensaje "Es muy poco", en caso de encontrar más de 2 pero menos de 4 elementos mostrará "No está mal" y en caso de otras posibilidades no enunciadas mostrará "Son Muchos!"

EJERCICIO3
- Dentro de una función llamada myFunction()
- Cree dos variables que contivieran cada una los valores ingresados por el usuario
- Cree otras dos variables que obtuvieran las secciones del html dónde deberán escribirse dichos valores
- Finalmente mediante el método innerHTML indiqué que los valores obtenidos de las primeras dos variables se escribieran en las secciones obtenidas del html en las variables de las líneas 24 y 25.
